\section{Explain the difference between == and ===}
In javascript == will check if the two values are equal. The === operator will also check if it has the same type. Example:

\begin{lstlisting}
0 == 0: True
0 == False: True
0 === 0: True
0 === False: False
\end{lstlisting}
